# Astrall RobotDog Cpp

Astrall SDK を使用して Astrall ロボット犬を制御するための C++17 デモプロジェクトです。

このプロジェクトでは、SDK の初期化、ロボットのテレメトリ購読、デバイス情報と電源状態の取得、ハートビート送信、キーボードによる移動制御を示します。

## 機能

- CMake ベースの C++ プロジェクト構成。
- `lib/astrall_sdk` による Astrall SDK 連携。
- キーボードによるスポーツモード、移動、認証、ライトの制御。
- IMU データとスポーツデータの購読。
- ハートビート、デバイス情報、システム状態、スポーツ状態、バッテリー状態の定期出力。
- ビルド後に SDK ランタイムライブラリを実行ファイルのディレクトリへ自動コピー。

## プロジェクト構成

```text
.
├── CMakeLists.txt
├── src/
│   ├── main.cpp
│   ├── keyboard.cpp
│   └── keyboard.h
├── lib/
│   └── astrall_sdk/
│       ├── include/
│       ├── x86-64/
│       └── arm64/
└── docs/
```

## 必要環境

- Linux 環境
- CMake 3.10 以降
- C++17 対応コンパイラ
- POSIX スレッド対応
- Astrall ロボット犬と互換性のある Astrall SDK ランタイム

## ビルド

```bash
cmake -S . -B build
cmake --build build
```

実行ファイルは `sdk_demo` として生成されます。必要な Astrall SDK ランタイムライブラリは、ビルド後に実行ファイルのディレクトリへコピーされます。

## 実行

```bash
./build/sdk_demo
```

ロボット環境に接続されたターミナルで実行してください。このデモは標準入力から直接キーボード入力を読み取ります。

## キーボード操作

| キー | 操作 |
| --- | --- |
| `1` | ダンピングモードへ切り替え |
| `2` | 固定スタンドモードへ切り替え |
| `3` | 固定ダウンモードへ切り替え |
| `4` | 移動モードへ切り替え |
| `5` | 自動充電を開始 |
| `6` | 充電を終了 |
| `9` | SDK 制御権限を要求 |
| `0` | 移動を停止 |
| `w` / `s` | 前後速度を増加 / 減少 |
| `a` / `d` | 横方向速度を増加 / 減少 |
| `q` / `e` | ヨー速度を増加 / 減少 |
| `n` | ライトをオン |
| `o` | ライトをオフ |

移動キーは、ロボットが移動モードの場合のみ有効です。

## 注意

このデモは安全で開けた場所で慎重に使用してください。移動を有効にする前に、SDK 制御権限があり、周囲に十分なスペースがあることを確認してください。
